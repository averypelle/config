# Config

This repo contains configurations that should be put into your home directory

## Dependencies

This repo depends on the following dependencies being installed

- git
- zsh
- oh-my-zsh
- alacritty
- nvm
- node
- pyenv
- pipenv
- python

## To install

Run the following commands

```bash
./sync.sh
```
